# html5-boilerplate

html5 boilerplate with gulp processes for hot reloading development environment.


## Instructions
```bash
$ yarn install

# development to build, server, and watch
$ yarn start

# just build
$ yarn build

# delete the /dist directory
$ yarn clean`

# prettify contents in the /src directory
$ yarn pretty
```
